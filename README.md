# コード採譜スタジオ

楽曲ファイルやステム分離MIDIから、**コード進行・ギターTAB譜・ベースTAB譜・鍵盤の五線譜(ドレミ表記)** を自動生成するWebツールです。すべての解析は端末内(ブラウザ内)で実行され、外部サーバーへの送信は一切ありません。

## 主な機能

### 🎵 音源ファイル(mp3 / wav / m4a など)
- キー(24調)・BPM・拍子の推定
- クロマグラム + Viterbi平滑化によるコード進行解析(参考精度)
- 再生と同期してコードがハイライト
- 使用コードのギターダイアグラム表示
- 移調(♯/♭)・ディグリーネーム表示

### 🎹 ステム分離MIDI(高精度採譜)
- 複数ファイル同時読み込みOK(楽器別ステム)
- トラックの楽器自動判別(手動変更も可能)
- **ギター** — 押さえるコードのダイアグラム + 6弦TAB譜(運指は自動最適化)
- **ベース** — 4弦TAB譜
- **鍵盤・メロディ** — 五線譜(お玉杓子)+ ドレミ表記
- コード進行 + ディグリーネーム
- 簡易シンセによる採譜結果の再生確認

### 📦 参考採譜の共有
- テキストTAB譜 (.txt) ダウンロード
- 譜面のPNG画像保存
- 印刷 / PDF保存(全パートまとめて)
- 「参考採譜パック」一括ダウンロード

## 高精度採譜のおすすめ手順

1. **ステム分離** — Sunoのステムダウンロード、Moises、Demucs などで楽器別に分離
2. **MIDI変換** — Basic Pitch、Melodyne、Klangio などで各ステムをMIDI化
3. **本ツールに読み込み** — MIDIファイルをまとめてドラッグ&ドロップ
4. ファイル名に `bass` `guitar` `piano` などが入っていると楽器を自動判別します

## ローカルでの確認方法

「ローカルで確認.command」をダブルクリック、またはターミナルで:

```bash
cd コード採譜スタジオ
python3 -m http.server 8942
# → http://localhost:8942 を開く
```

## 公開方法(無料)

静的サイトなので、フォルダごとアップロードするだけで公開できます。

### Netlify(いちばん簡単)
1. https://app.netlify.com/drop を開く
2. このフォルダをドラッグ&ドロップ → 即公開

### GitHub Pages
1. リポジトリを作成してこのフォルダの中身をアップロード
2. Settings → Pages → Branch を `main` にして保存

公開後、スマホで「ホーム画面に追加」すればアプリとして使えます(PWA・オフライン対応)。

## セキュリティ・プライバシー

- 音源・MIDIは**ブラウザ内だけ**で解析。外部送信なし
- Content-Security-Policy で外部リソース読み込みを全面遮断(完全自己完結)
- 外部ライブラリ・CDN・アナリティクス不使用

## ファイル構成

```
index.html            画面本体
style.css             デザイン
theory.js             音楽理論コア(コード判定・キー判定・運指・コードフォーム)
midi.js               MIDIパーサ & 高精度コード解析
audio.js              音源解析エンジン(FFT/クロマグラム/BPM/Viterbi)
render.js             SVG描画(ダイアグラム/TAB譜/五線譜/コードグリッド)
export.js             テキストTAB・採譜パック生成
app.js                UI制御・再生・書き出し
manifest.webmanifest  PWA設定
sw.js                 Service Worker(オフライン対応)
```
